Omega (State Density) Calculations

The omega calculation evaluates the two-component particle–hole state density \(\omega(p_\pi, h_\pi, p_\nu, h_\nu, E_x, Z, N)\) as defined in Eq. 13.4 of the TALYS manual. The implementation is in src/state_density.cpp and the main executable is src/main_omega.cpp.

Implementation

The state density is computed by particleHoleStateDensity() in src/state_density.cpp:17. The formula is:

\[ \omega(p_\pi, h_\pi, p_\nu, h_\nu, E_x, Z, N) = \frac{g_\pi^{p_\pi + h_\pi} g_\nu^{p_\nu + h_\nu}}{p_\pi! h_\pi! p_\nu! h_\nu! (n-1)!} \cdot E_x^{n-1} \cdot f_{\text{Pauli}} \cdot f_{\text{well}} \cdot f_{\text{pairing}} \cdot f_{\text{parity}} \]

where:

  • \(g_\pi = Z/15\), \(g_\nu = N/15\) — single-particle densities
  • \(n = p_\pi + h_\pi + p_\nu + h_\nu\) — total exciton number
  • \(f_{\text{Pauli}}\) — Pauli correction factor (pauliCorrection() in src/preeq_common.cpp)
  • \(f_{\text{well}}\) — finite well depth correction (finiteWell())
  • \(f_{\text{pairing}}\) — pairing correction (fuPairingCorrection())
  • \(f_{\text{parity}}\) — parity factor (default 1.0)

Running the Calculation

$ ./executables/omega

This enumerates all configurations with \(k = (p+h)/2\) from 0 to 5, producing 300 rows of output. The 13 columns are:

# Column Description
1–4 pπ, hπ, pν, hν Exciton state numbers
5 n Total exciton number (\(p_\pi + h_\pi + p_\nu + h_\nu\))
6 Ex Excitation energy [MeV]
7 U Available energy after pairing [MeV]
8 Δ Pairing energy [MeV]
9 Pph Pairing correction factor
10 ncrit Critical exciton number
11 gπ Proton single-particle density [MeV-1]
12 gν Neutron single-particle density [MeV-1]
13 ω Particle–hole state density [MeV-1]

Validation

The omega calculation can be validated against TALYS output:

$ ./executables/omega > omega_exciton.txt
$ python3 compare_omega.py omega_exciton.txt talys_omega.txt

The comparison script parses both 13-column YANDF-0.4 files and prints relative errors for each entry. See the Validations page for detailed results.

Fixed V vs. Energy-Dependent V

Two modes are tested in main_omega.cpp:

  • Fixed V = 38 MeV — uses constant well depth. This is the validated mode (max error 0.0085% against TALYS).
  • Energy-dependent V — uses potentialWellDepth() which decreases with excitation. This mode is not yet validated against TALYS.
