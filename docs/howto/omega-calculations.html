Omega — Particle-Hole State Density Calculations

How to compute, interpret, and compare particle-hole state densities with the omega executable.

Running the Omega Executable

$ cd /workspace/exciton
$ make executables/omega
$ ./executables/omega > omega_Ar40.txt

The omega executable has no CLI arguments. The target system is hardcoded as n + Ar-40 (Z=18, N=23, A=41 residual).

Understanding the Output

Block 1: State Densities (Ex = 1–25 MeV)

Columns:

ColumnDescriptionUnits
ExExcitation energyMeV
P(3)Pairing correction energy \(\Delta\)MeV
g(p)Single-particle state density (proton)MeV-1
g(n)Single-particle state density (neutron)MeV-1
1p1h0p0h … 1p1h1p1hState density \(\omega(p_\pi, h_\pi, p_\nu, h_\nu, E_x)\) for 9 exciton configurationsMeV-1

The 9 exciton configurations computed are:

\[ \begin{aligned} \omega(1,1,0,0),\; \omega(0,0,1,1),\; \omega(1,1,1,0),\; \omega(1,0,1,1),\; \\ \omega(2,1,0,0),\; \omega(0,0,2,1),\; \omega(2,2,0,0),\; \omega(0,0,2,2),\; \omega(1,1,1,1) \end{aligned} \]

Block 2: Spin Distribution (n = 1–12)

Columns: n, J=0 through J=8, Sum (summed to J=30).

The spin distribution formula (preeq_common.cpp:148):

\[ R_n(J) = \frac{2J+1}{\sqrt{\pi}\, k^{3/2}\, \sigma^3} \exp\!\left[-\frac{(J+\tfrac12)^2}{k\,\sigma^2}\right], \quad \sigma^2 = 0.24\, n\, A^{2/3} \]

where \(k=2\) in TALYS-compatibility mode. Parity is 50/50: \(\omega(p,h,E,J,\pi) = \omega(p,h,E) \times R_n(J) \times \tfrac12\).

Comparing Omega Outputs

$ python3 compare_omega.py file1.txt file2.txt report.txt
$ python3 compare_omega.py file1.txt file2.txt report.txt 1e-6

The script compares both data blocks and produces absolute differences and relative percentages. Default tolerance: 10-4 absolute / 1% relative.

Typical Results

The omega calculation for Ar-40 shows:

  • Single-particle state densities: \(g_p = 18/15 = 1.2\) MeV-1, \(g_n = 23/15 \approx 1.533\) MeV-1
  • Pairing energy: \(\Delta = 1 \times 12/\sqrt{41} \approx 1.874\) MeV (odd-A target, chi = 1)
  • State densities grow rapidly with Ex and exciton number n, typical of combinatorial phase space

For validation against TALYS, the code uses a fixed well depth V = 38.0 MeV (not the energy-dependent formula from Eq. 13.11) to match the TALYS reference calculation.

Troubleshooting

Omega values are zero at low Ex: This is expected when the excitation energy is below the Pauli blocking threshold for a given exciton configuration. The minimum energy required for configuration \((p_\pi, h_\pi, p_\nu, h_\nu)\) is \(\max(p_\pi, h_\pi)^2/g_p + \max(p_\nu, h_\nu)^2/g_n\).
