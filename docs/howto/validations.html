Validations

This project validates against TALYS 2.2 output for the 40Ar(n,x) system. Two Python comparison scripts are provided.

1. State Density (Omega) Validation

Script: compare_omega.py

Parses two 13-column YANDF-0.4 files and compares all 300 (pπ,hπ,pν,hν) configurations.

$ python3 compare_omega.py exciton_omega.txt talys_omega.txt
Test Case Result Max Error
Fixed V=38 MeV PASS 0.0085% relative (all 300 values)
Energy-dependent V FAIL Large deviations — not yet matched to TALYS

2. Spin Distribution Validation

The spin distribution function spinDistribution(J, A, n) in src/preeq_common.cpp computes the fraction of states with total spin J for an n-exciton configuration. Validated for J=0..8 and n=1..12:

Test Case Result Max Error
All 120 (J,n) pairs PASS 0.0095% relative

3. Transition Rate (Lambda) Validation

Script: compare_lambda.py

Parses two 8-column YANDF-0.4 files and compares all 84 exciton states (k=0..5).

$ python3 compare_lambda.py exciton_lambda.txt talys_lambda.txt

4. Running the Full Validation Suite

$ # Omega (state densities)
./executables/omega > omega_exciton.txt
python3 compare_omega.py omega_exciton.txt path/to/talys_omega.txt

$ # Lambda analytical
./executables/lambda --analytical > lambda_analytical.txt
python3 compare_lambda.py lambda_analytical.txt path/to/talys_analytical.txt

$ # Lambda numerical (guarded Clenshaw-Curtis)
./executables/lambda > lambda_numerical.txt
python3 compare_lambda.py lambda_numerical.txt path/to/talys_numerical.txt
