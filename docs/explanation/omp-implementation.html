Optical Model Potential Implementation

Purpose: Replace the constant M² matrix element with a collision probability derived from the Koning–Delaroche (KD03) global optical model. This page is a summary; the full deep-dive covers the complete data flow, formulas, and code map.

Overview

The OMP implementation resides on the OMP branch and implements TALYS preeqmode=3. The flow is:

  1. CLI flag --kernel optical-model selects the OMP collision kernel
  2. lambdaRate() dispatches to the OMP integrand branches
  3. For each energy bin e in the numerical integration:
    1. Compute compound nucleus (Zcomp, Acomp) = (Z+Zproj, A+Ap)
    2. Load separation energies from MARLEY MassTable
    3. Compute ekin = e − Sp/n(Zcomp, Ncomp)
    4. Instantiate KoningDelarocheOpticalModel with (Zcomp, Acomp)
    5. Call setIncidentEnergyAndFragment(e_kin, collider_pdg) with two_s=0 (central potential only)
    6. Call wvolRadialIntegral() for volume-averaged ⟨W⟩
    7. Collision rate = (2/ℏ) · wompfac · ⟨W⟩
    8. Multiply by state densities ωt · ωr

Key Components

wompfac

TALYS weighting factors bridging the M² and OMP normalizations. Three values are computed:

  • wompfac_same — used for same-type pair creation (ππ, νν)
  • wompfac_cross — used for cross-type pair creation (πν, νπ)
  • wompfac_conv — used for conversion channels (π↔ν)

The OMP constant COMP = 0.55 is from TALYS matrix.f90. At default parameters (M²c = 1.0, Rπν = 1.0): wompfacsame = 0.1833, wompfaccross = 0.3667, wompfacconv = 0.275.

Koning–Delaroche Optical Model

MARLEY's KoningDelarocheOpticalModel provides the imaginary potential parameters Wv, Wd, Rv, av, Rd, ad as functions of incident nucleon energy. All KD03 parameters, well depths, and radial potentials U(r) have been validated against TALYS output.

Volume-Averaged ⟨W⟩

Eq. 13.30 with a 50-bin midpoint rule (dr = 0.4 fm), matching TALYS bonetti.f90. The radial profile is:

\[ W(r) = W_v f_v(r) - 4 a_d W_d \frac{d}{dr} f_s(r) \]

where fv and fs are Woods–Saxon form factors.

MARLEY MassTable

Separation energies Sp, Sn come from the MARLEY MassTable singleton, using AME experimental masses with Koning liquid-drop fallback. The reference nucleus is the compound nucleus.
