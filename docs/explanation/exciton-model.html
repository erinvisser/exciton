Explanation

The Two-Component Exciton Model

The exciton model is a semi-classical, statistical description of pre-equilibrium nuclear reactions. It describes how a nuclear reaction evolves from a simple initial configuration (typically a single particle and hole) toward the fully equilibrated compound nucleus through a sequence of internal transitions that create or annihilate particle-hole pairs.

Key Idea

After the first stage of a nuclear reaction (direct reactions, which happen on the timescale of the nuclear transit time ∼10-22 s), the system is left in a state with a few "excitons" — particles excited above the Fermi energy and holes below it. The exciton model tracks the statistical population of these configurations using Fermi's Golden Rule.

Two-Component Formalism

The two-component extension (Kalbach, 1986) separates protons and neutrons explicitly. An exciton state is described by four integers:

\[ (p_\pi, h_\pi, p_\nu, h_\nu) \]

where \(p\) = particle number, \(h\) = hole number, \(\pi\) = proton, \(\nu\) = neutron. The total exciton number is:

\[ n = p_\pi + h_\pi + p_\nu + h_\nu \]

Internal Transition Types

There are four fundamental transition types in the two-component model:

TypeProcessEffect
Proton pair creation \(\pi + \nu \to \pi + \pi' + \pi^{-1} + \nu\) \(p_\pi \to p_\pi+1,\; h_\pi \to h_\pi+1\)
Neutron pair creation \(\pi + \nu \to \pi + \nu' + \nu^{-1} + \nu\) \(p_\nu \to p_\nu+1,\; h_\nu \to h_\nu+1\)
Proton-to-neutron conversion \(\pi + \nu \to \nu + \nu' + \pi^{-1}\) \(p_\pi \to p_\pi-1,\; h_\pi \to h_\pi-1,\; p_\nu \to p_\nu+1,\; h_\nu \to h_\nu+1\)
Neutron-to-proton conversion \(\pi + \nu \to \pi + \pi' + \nu^{-1}\) \(p_\nu \to p_\nu-1,\; h_\nu \to h_\nu-1,\; p_\pi \to p_\pi+1,\; h_\pi \to h_\pi+1\)

The Master Equation

The time evolution of the exciton configuration population \(q(p_\pi, h_\pi, p_\nu, h_\nu, t)\) is governed by:

\[ \frac{dq}{dt} = \lambda^+ q_{n-2} + \lambda^- q_{n+2} + \lambda^{\text{conv}} q_{n}^{\text{conv}} - (\lambda^+ + \lambda^- + \lambda^{\text{conv}}) q \]

The exciton code computes the internal transition rates \(\lambda^+\) (pair creation), \(\lambda^-\) (pair annihilation), and \(\lambda^{\text{conv}}\) (conversion) using either:

  • Constant matrix element (M²) approximation (Kalbach-Cline parametrization, TALYS preeqmode=1)
  • Optical model potential (OMP) collision rates (Koning-Delaroche KD03, TALYS preeqmode=3)

Relation to TALYS

This code is designed to reproduce TALYS pre-equilibrium calculations. The key references are:

  • TALYS 1.96 manual, Section 13 — Two-component exciton model
  • Kalbach (1986), PRC 33, 818 — Original two-component formalism
  • Koning & Delaroche (2003), NPA 713, 231 — Global optical model
  • Ignatyuk et al. (1975) — State density parameterization
Limitations: The current code assumes a single compound nucleus (Z=18, N=22, A=40 for lambda; Z=18, N=23, A=41 for omega) and uses a fixed Fermi well depth V = 38 MeV. It does not yet compute particle emission cross sections — only the internal transition rates that are the input to the master equation.
