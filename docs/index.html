exciton Documentation

A C++ implementation of the two-component exciton model for pre-equilibrium nuclear reactions, validating against TALYS preeqmode=3. Created by Erin Visser.

Repository: /workspace/exciton — Dependency: MARLEY library at ../marley-private/

Quick Reference

WhatCommandOutput
Build everything make Executables omega and lambda
Omega (state densities) make run-omega YANDF-0.4 state densities + spin distributions
Lambda (rates, matrix element) make run ARGS="--kernel matrix-element" YANDF-0.4 transition rates
Lambda (rates, OMP kernel) make run ARGS="--kernel optical-model" YANDF-0.4 transition rates + KD03 diagnostics
Validate KD03 parameters make run ARGS="--mode kd03" KD03 parameters log + Q-values + rates
Validate well depths make run ARGS="--mode well_depths" Well depths log
Validate radial potential make run ARGS="--mode radial_potential" Radial OMP log
Compare lambda outputs python3 compare_lambda.py f1 f2 report [tol] Absolute + relative error report
Compare omega outputs python3 compare_omega.py f1 f2 report [tol] Absolute + relative error report
TALYS OMP extraction python3 talys_total_omp/talys_omp_total.py --energies 20.0 TALYS OMP radial profiles
